# 20260523234500-修复历史活动表格对齐问题

## 问题提示词

用户反馈：抽奖后台的历史活动中的tab中参与人数和下面的数字没有对齐

## 问题分析

表格中"参与人数"列的表头和数据单元格对齐不一致，原因是：
1. 表头默认左对齐
2. 数据单元格中的 `<span class="link-text">` 元素没有统一的对齐方式

## 解决方案

### 1. 修改 link-text 样式

添加 `display: inline-block`、`min-width: 40px` 和 `text-align: center` 属性，使链接文本居中对齐并有固定最小宽度：

```css
.link-text {
  color: #3498db;
  cursor: pointer;
  text-decoration: underline;
  display: inline-block;
  min-width: 40px;
  text-align: center;
}
```

### 2. 修改表格第三列对齐方式

使用 `nth-child` 选择器让"参与人数"列（第三列）居中对齐：

```css
.history-table th:nth-child(3),
.history-table td:nth-child(3) {
  text-align: center;
}
```

## 修改文件

- `lucky-admin/src/App.vue` - 修改表格样式

1. ✅ "参与人数"表头居中对齐
2. ✅ 参与人数数字居中对齐
3. ✅ 表头和数据单元格对齐一致
