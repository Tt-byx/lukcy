# 大屏页面 — 移除活动名称 + 动态背景

## 用户问题

修改 `lucky-screen\src\App.vue`，完成以下变更：

1. **移除顶部活动名称**：删除 header 中的 `<h1 class="title">`，stats 右对齐
2. **删除 activityName 相关代码**：移除 `activityName` ref、`fetchActivityInfo` 函数，替换为精简版 `fetchParticipantCount`
3. **新增动态背景功能**：支持视频/图片背景切换，通过 `/api/screen/config` 接口获取配置，每 30 秒轮询

## 分析过程

### 2. 变更方案设计

| 模块 | 操作 | 原因 |
|------|------|------|
| 模板 header | 删除 `<h1.title>`，stats 加 `margin-left: auto` | 用户要求移除活动名称 |
| 模板 screen | 添加 `:style="screenBackgroundStyle"` | 支持图片背景 inline style 覆盖 |
| 模板背景层 | 在 bg-particles 前插入 `<div v-if="...">` | 视频背景需要独立的 video 标签 |
| 脚本导入 | 添加 `computed` | screenBackgroundStyle 需要响应式计算 |
| 脚本状态 | 删除 `activityName`，添加 `screenConfig` `bgPollTimer` | 移除旧功能，添加新状态 |
| 脚本函数 | `fetchActivityInfo` → `fetchParticipantCount` | 只保留参与人数获取逻辑 |
| 脚本函数 | 新增 `fetchScreenConfig` `getBgUrl` | 屏幕配置获取和背景 URL 拼接 |
| 脚本计算 | 新增 `screenBackgroundStyle` computed | 图片背景时动态生成 CSS |
| 生命周期 | onMounted 添加轮询，onUnmounted 清理 | 30 秒刷新背景配置 |
| CSS | `.header` 改 flex-end + z-index:2 | 右对齐 + 层级 |
| CSS | `.bg-particles` 加 z-index:1 | 确保粒子在背景之上 |
| CSS | 新增 `.bg-video-wrapper` `.bg-video` | 视频全屏覆盖样式 |

### 3. Z-index 层级设计
```
z-index: 0     → 视频背景层
z-index: 1     → 粒子效果
z-index: 2     → 顶部信息栏
z-index: 10    → 抽奖滚动动画
z-index: 20    → 中奖结果展示
z-index: 100   → 弹幕层
z-index: 999   → 弹幕设置面板
z-index: 1000  → 弹幕设置按钮
```
