# 历史活动功能 + 移除抽奖管理 tab

## 用户问题
1. 抽奖活动结束后，抽奖管理里就没有记录了，希望加一个历史活动目录，点进去能看到历史的抽奖活动和中奖人
2. 抽奖管理这个 tab 没必要了，活动管理里的快速抽奖更好用

## 修改内容

### 后端（3 个文件）

#### ActivityService.java
新增方法签名：
```java
List<Activity> getHistoryActivities();
```

#### ActivityServiceImpl.java
新增实现：查询 status=2（已结束）的活动，按创建时间倒序：
```java
@Override
public List<Activity> getHistoryActivities() {
    return list(new LambdaQueryWrapper<Activity>()
            .eq(Activity::getStatus, 2)
            .orderByDesc(Activity::getCreatedAt));
}
```

#### ActivityController.java
新增接口 `GET /api/activity/history`：
```java
@GetMapping("/history")
public Result<List<Activity>> getHistory() {
    return Result.ok(activityService.getHistoryActivities());
}
```

> 复用已有接口：`GET /api/lottery/rounds?activityId=` 和 `GET /api/lottery/winners?roundId=`

## 功能说明

### 侧边栏
```
├── 活动管理
├── 历史活动    ← 新增
├── 弹幕审核
└── 敏感词
```

### 历史活动页交互
- 进入页面自动加载所有已结束的活动
- 点击活动行可展开/收起详情（箭头指示）
- 展开后自动加载该活动的所有轮次和每个轮次的中奖者
- 中奖者以标签形式展示，带蓝色底色

### 边界处理
- 无历史活动 → "暂无历史活动"
- 活动无轮次 → "该活动没有抽奖记录"
- 轮次无中奖者 → 不显示中奖者区域

## 用户需要做的
1. **重启后端**：`mvn spring-boot:run`（新接口生效）
2. **重启 lucky-admin 前端**：`npm run dev`
3. 活动结束后，点击"历史活动"即可查看
