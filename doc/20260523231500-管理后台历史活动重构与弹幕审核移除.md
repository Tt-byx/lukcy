# 20260523231500-管理后台历史活动重构与弹幕审核移除

## 问题提示词

用户要求：
1. 后台管理页面中不需要弹幕审核的tab，因为根本用不上
2. 历史活动希望大改一下界面，用一个表格的格式，表头是活动名、创建时间、参与人数、操作
3. 操作表头下面有删除、详情的按钮
4. 删除后就没有这个活动记录
5. 点击详情后可以看到该活动的参与人数、获奖情况等

## 分析过程

### 1. 代码结构分析

管理后台代码位于 `lucky-admin/src/App.vue`，采用 Vue 3 + Composition API 构建。

主要功能模块：
- 活动管理
- 历史活动
- 弹幕审核
- 敏感词管理
- 屏幕设置
- 用户管理

### 2. 后端API分析

- 删除活动API需要新增：`DELETE /api/activity/{id}`
- 获取参与人数API已存在：`GET /api/participant/count?activityId=xxx`
- 获取抽奖轮次API已存在：`GET /api/lottery/rounds?activityId=xxx`
- 获取中奖者API已存在：`GET /api/lottery/winners?roundId=xxx`

## 解决方案

### 1. 后端修改

在 `ActivityController.java` 中添加删除活动的API：

```java
@DeleteMapping("/{id}")
public Result<Void> delete(@PathVariable Long id) {
    activityService.removeById(id);
    return Result.ok();
}
```

### 2. 前端修改

#### 2.1 移除弹幕审核Tab

删除侧边栏中的弹幕审核导航链接和对应的面板内容。

#### 2.2 重构历史活动界面

将原有的折叠式卡片布局改为表格布局：

```html
<table class="history-table">
  <thead>
    <tr>
      <th>活动名</th>
      <th>创建时间</th>
      <th>参与人数</th>
      <th>操作</th>
    </tr>
  </thead>
  <tbody>
    <tr v-for="act in historyActivities" :key="act.id">
      <td>{{ act.name }}</td>
      <td>{{ act.createdAt?.substring(0, 16) }}</td>
      <td>{{ act.participantCount || 0 }}</td>
      <td class="action-cell">
        <button class="btn-sm btn-info" @click="showHistoryDetail(act)">详情</button>
        <button class="btn-sm btn-danger" @click="deleteActivity(act)">删除</button>
      </td>
    </tr>
  </tbody>
</table>
```

#### 2.3 实现详情弹窗

添加模态框显示活动详情，包括：
- 参与人数统计
- 创建时间
- 抽奖轮次列表
- 每个轮次的中奖者

#### 2.4 实现删除功能

添加删除活动功能，包含确认对话框，删除后自动刷新列表。

### 3. 新增状态变量

```javascript
// 详情弹窗相关
const showDetailModal = ref(false)
const detailActivity = ref(null)
const detailRounds = ref([])
const detailWinners = ref({})
```

### 4. 新增函数

- `deleteActivity(activity)` - 删除活动
- `showHistoryDetail(activity)` - 显示活动详情弹窗

### 5. 新增CSS样式

- `.history-table` - 历史活动表格样式
- `.modal-overlay` - 弹窗遮罩层
- `.modal-content` - 弹窗内容区
- `.detail-stats` - 统计数据展示
- `.round-detail` - 轮次详情卡片
- `.winner-tag` - 中奖者标签
