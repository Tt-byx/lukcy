# 弹幕全屏改造 + 活动进行中抽奖入口

## 用户问题
1. 弹幕只在右半边屏幕显示，希望参考B站弹幕设置，可以自由配置
2. 只有活动开始/结束功能，活动进行中需要一个抽奖功能

## 修改内容

### 一、弹幕全屏改造（lucky-screen/src/App.vue）

#### 1. 弹幕层移出右面板，改为全屏覆盖
- 弹幕层从 `right-panel` 内部移出，放在 `.screen` 顶层
- `position: absolute` → `position: fixed`，覆盖整个屏幕
- 弹幕从屏幕最右侧开始，可覆盖二维码区域上方

#### 2. 新增弹幕设置面板（参考B站）
右上角齿轮按钮 ⚙ 点击弹出设置面板，可配置：
| 设置项 | 说明 | 范围 |
|--------|------|------|
| 显示区域 | 全屏 / 上半屏 / 下半屏 | 三选一 |
| 不透明度 | 弹幕文字透明度 | 20%-100% |
| 字号大小 | 弹幕字体基准大小 | 16px-48px |
| 滚动速度 | 弹幕从右到左的时间 | 4s-16s |

#### 3. localStorage 持久化
- 设置自动保存到 `localStorage`
- 页面刷新后自动恢复上次的设置
- 使用 `watch` 监听设置变化自动保存

#### 4. 弹幕效果优化
- 颜色从7种增加到10种
- 字号随机波动 ±4px，速度随机波动 ±30%，更自然
- 弹幕从 `left: 100%` 开始（原来 `right: -100%` 在 fixed 布局下有问题）

### 二、活动管理页增加抽奖入口（lucky-admin/src/App.vue）

#### 修改内容
- 活动状态为"进行中"（status === 1）时，在当前活动卡片下方显示**快速抽奖**卡片
- 包含：创建轮次表单、轮次列表（含开始抽奖按钮）、中奖记录
- 切换到"活动管理"tab 时自动加载轮次列表
- 原有的"抽奖管理"tab 保持不变，两个入口均可使用

## 结果
- 弹幕现在全屏覆盖，右上角齿轮可自由调节区域、透明度、字号、速度
- 设置会自动保存，刷新不丢失
- 活动进行中时，直接在活动管理页就能创建轮次并抽奖，无需切换tab
