1. 侧边栏新增 Tab
2. 屏幕设置面板（当前背景预览、自定义背景上传、恢复默认）
3. state 变量和方法
4. watch 切换加载
5. CSS 样式

## 分析过程

### 1. 需求分析
需要在前端管理后台新增一个"屏幕设置"Tab，用于管理大屏的显示背景。功能包括：
- 查看当前使用的背景（图片/视频预览）
- 上传自定义背景（支持图片和视频）
- 清除自定义背景恢复默认
- 对应的后端 API：GET `/api/screen/config`、POST `/api/screen/background`、DELETE `/api/screen/background`

### 2. 代码分析
首先读取 `App.vue` 完整文件，了解现有结构：
- 模板层：侧边栏导航 + 4 个面板（活动管理、历史活动、弹幕审核、敏感词管理）
- script setup：使用 Vue 3 Composition API（ref、watch、onMounted），原生 fetch API 调用后端
- 样式：scoped scss，2 空格缩进

### 3. 关键设计决策
- **fullUrl 预计算**：模板中无法直接访问 `API_BASE` 变量（只有 script 中定义），所以在 `fetchScreenConfig` 和 `uploadBackground` 的响应处理中将 `API_BASE + backgroundUrl` 拼接为 `fullUrl` 字段存入 `screenConfig` 对象，模板中直接使用 `screenConfig.fullUrl`。
- 保持现有代码风格：2 空格缩进、中文文案、原生 fetch API、`async/await` 错误处理模式。
