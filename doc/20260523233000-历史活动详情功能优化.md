# 20260523233000-历史活动详情功能优化

## 问题提示词

用户要求：
1. 创建时间变为年月日时分秒，不要中间的T
2. 参与人数希望也可以查看详细，可以看看谁参与了，什么时候参与了，本次活动发了什么弹幕
3. 显示的是姓名，不是参与者1、参与者2这种数字

## 分析过程

### 1. 时间格式问题

原代码使用 `substring(0, 16)` 截取时间字符串，会保留中间的T字符。需要改为使用 JavaScript 的 Date 对象格式化时间。

### 2. 参与者详情功能

需要新增：
- 参与者列表弹窗，显示所有参与者的姓名、学号、参与时间、弹幕数
- 弹幕详情弹窗，显示某个参与者发送的所有弹幕

### 3. 中奖者姓名显示

Winner 实体中只有 participantId，需要通过 participantId 查询参与者姓名。

## 解决方案

### 1. 时间格式化函数

```javascript
function formatTime(time) {
  if (!time) return '-'
  const date = new Date(time)
  const year = date.getFullYear()
  const month = String(date.getMonth() + 1).padStart(2, '0')
  const day = String(date.getDate()).padStart(2, '0')
  const hours = String(date.getHours()).padStart(2, '0')
  const minutes = String(date.getMinutes()).padStart(2, '0')
  const seconds = String(date.getSeconds()).padStart(2, '0')
  return `${year}-${month}-${day} ${hours}:${minutes}:${seconds}`
}
```

### 3. 中奖者姓名显示

在 `showHistoryDetail` 函数中，获取中奖者后，通过 participantId 查询参与者姓名：
```javascript
for (const winner of winners) {
  if (winner.participantId && !winner.name) {
    const participantRes = await fetch(`${API_BASE}/api/participant/list?activityId=${activity.id}`)
    const participantData = await participantRes.json()
    if (participantData.code === 200) {
      const participant = participantData.data.find(p => p.id === winner.participantId)
      if (participant) {
        winner.name = participant.name
      }
    }
    break
  }
}
```
