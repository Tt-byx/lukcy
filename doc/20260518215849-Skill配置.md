# Skill配置

## 本次问答的提示词

用户要求将 `C:\Users\28182\.claude\skills` 目录下的skill配置到当前项目中。

## 做了些什么事情

1. 查看 `C:\Users\28182\.claude\skills` 目录内容，发现有一个 `frontend-design.md` 文件
2. 读取 `frontend-design.md` 文件内容，了解其功能
3. 在项目中创建 `.claude/skills/` 目录
4. 将 `frontend-design.md` 复制到项目的 `.claude/skills/` 目录下

## 为什么这样做

Claude Code的skill文件需要放在项目的 `.claude/skills/` 目录下才能被项目识别和使用。通过将skill文件复制到项目中，可以使该项目在Claude Code会话中自动加载该skill。

## 产生了什么样的结果

成功将 `frontend-design` skill配置到项目中。该skill的功能是：

- **名称**: frontend-design
- **用途**: 创建独特的、生产级的前端界面，具有高质量设计
- **适用场景**: 当用户要求构建Web组件、页面、应用程序、海报等时使用
- **特点**: 生成创意、精美的代码和UI设计，避免通用的AI美学风格

### Skill包含的设计指导

1. **设计思维**: 在编码前理解上下文，选择大胆的美学方向
2. **前端美学指南**:
   - 排版：选择美丽、独特、有趣的字体
   - 颜色与主题：使用CSS变量保持一致性
   - 动画：使用动画效果和微交互
   - 空间构图：非常规布局、不对称、重叠
   - 背景与视觉细节：创建氛围和深度
